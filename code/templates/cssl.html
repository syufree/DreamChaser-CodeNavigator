CSS 基础教程

CSS 简介

CSS（Cascading Style Sheets）是一种样式表语言，用于描述 HTML 或 XML 文档的呈现方式。

body {
    background-color: lightblue;
}

选择器

选择器用于选取你想要设置样式的 HTML 元素。

p {
    color: blue;
}

脚注

脚注用于选取你想要设置样式的 HTML 元素。


footer{color：red；}

盒子模型

CSS 盒子模型描述了元素的内容、内边距、边框和外边距。

div {
    width: 100px;
    padding: 20px;
    border: 5px solid black;
    margin: 10px;
}

Flexbox

Flexbox 是一种布局模式，用于一维布局。

.container {
    display: flex;
    justify-content: space-between;
}
