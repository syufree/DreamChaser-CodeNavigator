JavaScript 基础教程

JavaScript 简介

JavaScript 是一种轻量级的解释型或即时编译型编程语言，具有函数优先的编程风格，是一种广泛用于客户端和服务端的编程语言。

console.log("Hello, World!");

变量

在 JavaScript 中，可以使用 var、let 和 const 关键字声明变量。

let myVariable = 10;
console.log(myVariable);

条件语句

条件语句根据条件的真假执行不同的代码块。常见的条件语句有 if、else if 和 else。

let x = 10;
if (x > 0) {
    console.log("x is positive");
} else if (x == 0) {
    console.log("x is zero");
} else {
    console.log("x is negative");
}

循环

循环用于重复执行某段代码。JavaScript 支持 for 循环和 while 循环。

for (let i = 0; i < 5; i++) {
    console.log(i);
}

函数

函数是执行特定任务的一段代码。可以使用 return 关键字返回值。

function add(a, b) {
    return a + b;
}

console.log(add(5, 3));
