C++ 介绍

C++ 是一种广泛使用的编程语言，具有面向对象、泛型编程和低级内存操作等特性。

语法基础

C++ 语法包括变量声明、函数定义、控制结构等。

#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}

变量和数据类型

C++ 支持多种数据类型，如整数、浮点数、字符和布尔值。

int age = 25;
float height = 5.9;
char initial = 'A';
bool is_student = true;

控制结构

C++ 支持常见的控制结构，如 if-else 语句、for 循环和 while 循环。

if (age > 18) {
    std::cout << "Adult" << std::endl;
} else {
    std::cout << "Minor" << std::endl;
}

for (int i = 0; i < 5; i++) {
    std::cout << i << std::endl;
}

while (age < 30) {
    age++;
}

函数

函数用于组织代码，提高代码复用性。

int add(int a, int b) {
    return a + b;
}

int main() {
    int result = add(3, 4);
    std::cout << "Result: " << result << std::endl;
    return 0;
}

类和对象

类是C++的基本构建模块，包含数据和操作数据的函数。

class Person {
public:
    std::string name;
    int age;

    void greet() {
        std::cout << "Hello, my name is " << name << std::endl;
    }
};

int main() {
    Person person;
    person.name = "Alice";
    person.age = 30;
    person.greet();
    return 0;
}

继承

继承用于创建一个类，该类从另一个类继承属性和方法。

class Animal {
public:
    void speak() {
        std::cout << "Animal speaks" << std::endl;
    }
};

class Dog : public Animal {
public:
    void bark() {
        std::cout << "Dog barks" << std::endl;
    }
};

int main() {
    Dog dog;
    dog.speak();
    dog.bark();
    return 0;
}

多态

多态允许用同一个接口调用不同的实现。

class Base {
public:
    virtual void show() {
        std::cout << "Base class" << std::endl;
    }
};

class Derived : public Base {
public:
    void show() override {
        std::cout << "Derived class" << std::endl;
    }
};

int main() {
    Base* basePtr;
    Derived derived;
    basePtr = &derived;
    basePtr->show();
    return 0;
}

模板

模板允许编写泛型函数和类。

template <typename T>
T add(T a, T b) {
    return a + b;
}

int main() {
    std::cout << add(3, 4) << std::endl;
    std::cout << add(3.5, 4.5) << std::endl;
    return 0;
}

标准模板库（STL）

STL 包含常用的数据结构和算法。

#include <iostream>
#include <vector>

int main() {
    std::vector<int> vec = {1, 2, 3, 4, 5};
    for (int val : vec) {
        std::cout << val << " ";
    }
    return 0;
}

异常处理

异常处理用于捕获和处理运行时错误，避免程序崩溃。

try {
    int result = 10 / 0;
} catch (std::exception& e) {
    std::cerr << "Exception: " << e.what() << std::endl;
}

文件输入输出

C++ 提供了简单的文件读写操作，可以用 ifstream 和 ofstream 类。

#include <fstream>

int main() {
    std::ofstream outfile("file.txt");
    outfile << "Hello, World!" << std::endl;
    outfile.close();

    std::ifstream infile("file.txt");
    std::string content;
    std::getline(infile, content);
    std::cout << content << std::endl;
    infile.close();
    return 0;
}

指针

指针用于存储变量的地址。

int main() {
    int a = 10;
    int* ptr = &a;
    std::cout << "Value of a: " << a << std::endl;
    std::cout << "Address of a: " << ptr << std::endl;
    std::cout << "Value at address: " << *ptr << std::endl;
    return 0;
}

引用

引用是变量的别名，操作引用就像操作原变量。

int main() {
    int a = 10;
    int& ref = a;
    ref = 20;
    std::cout << "Value of a: " << a << std::endl;
    return 0;
}

命名空间

命名空间用于防止命名冲突。

namespace MyNamespace {
    void display() {
        std::cout << "Hello from MyNamespace" << std::endl;
    }
}

int main() {
    MyNamespace::display();
    return 0;
}

预处理器

预处理器用于处理包含文件、宏定义等。

#define PI 3.14

int main() {
    std::cout << "Value of PI: " << PI << std::endl;
    return 0;
}

多线程

C++11 提供了多线程编程支持。

#include <thread>

void print_numbers() {
    for (int i = 0; i < 5; i++) {
        std::cout << i << std::endl;
    }
}

int main() {
    std::thread t1(print_numbers);
    t1.join();
    return 0;
}

Lambda 表达式

Lambda 表达式用于定义匿名函数。

int main() {
    auto add = [](int a, int b) {
        return a + b;
    };
    std::cout << add(3, 4) << std::endl;
    return 0;
}

智能指针

智能指针用于自动管理内存。

#include <memory>

int main() {
    std::unique_ptr<int> ptr(new int(10));
    std::cout << "Value: " << *ptr << std::endl;
    return 0;
}

移动语义

移动语义用于高效地转移资源。

#include <vector>

int main() {
    std::vector<int> vec1 = {1, 2, 3, 4, 5};
    std::vector<int> vec2 = std::move(vec1);
    for (int val : vec2) {
        std::cout << val << " ";
    }
    return 0;
}
