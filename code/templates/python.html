Python 介绍

Python 是一种以其简单性和可读性著称的流行编程语言。它广泛应用于Web开发、数据分析、人工智能、科学计算等领域。

变量和数据类型

在Python中，变量不需要声明类型，可以直接赋值。常见的数据类型有整数、浮点数、字符串和布尔值。

age = 25
name = "Alice"
height = 5.7
is_student = True

控制结构

Python 支持常见的控制结构，如 if-else 语句、for 循环和 while 循环。

if age > 18:
    print("Adult")
else:
    print("Minor")

for i in range(5):
    print(i)

while age < 30:
    age += 1

函数和模块

函数用于组织代码，提高代码复用性。模块是包含函数和变量的文件，可以导入到其他文件中使用。

def greet(name):
    return f"Hello, {name}"

import math
print(math.sqrt(16))

文件处理

Python 提供了简单的文件读写操作，可以用 open() 函数打开文件，用 read() 或 write() 函数进行读写。

with open('file.txt', 'r') as file:
    content = file.read()

with open('file.txt', 'w') as file:
    file.write("Hello, World!")

异常处理

异常处理用于捕获和处理运行时错误，避免程序崩溃。常用的语句有 try、except 和 finally。

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")
finally:
    print("Execution complete")

面向对象编程

Python 支持面向对象编程，可以定义类和创建对象。

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hello, my name is {self.name}"

person = Person("Alice", 30)
print(person.greet())

装饰器

装饰器用于在不修改原函数的情况下扩展函数的功能。

def my_decorator(func):
    def wrapper():
        print("Something is happening before the function is called.")
        func()
        print("Something is happening after the function is called.")
    return wrapper

@my_decorator
def say_hello():
    print("Hello!")

say_hello()

生成器

生成器用于生成一个值序列，每次调用时生成一个值。使用 yield 关键字。

def count_up_to(max):
    count = 1
    while count <= max:
        yield count
        count += 1

counter = count_up_to(5)
print(next(counter))
print(next(counter))

迭代器

迭代器是实现了迭代协议的对象，可以使用 iter() 和 next() 函数遍历。

my_list = [1, 2, 3]
iterator = iter(my_list)
print(next(iterator))
print(next(iterator))

标准库和第三方库

Python 提供了丰富的标准库和第三方库，可以用来简化各种任务。

import os
print(os.getcwd())

import requests
response = requests.get('https://api.github.com')
print(response.status_code)

正则表达式

正则表达式用于匹配字符串中的特定模式。

import re
pattern = r'\d+'
text = 'My phone number is 123-456-7890'
matches = re.findall(pattern, text)
print(matches)

列表生成式

列表生成式是一种简洁的创建列表的方式。

squares = [x**2 for x in range(10)]
print(squares)

上下文管理器

上下文管理器用于管理资源，如文件、网络连接等。

with open('file.txt', 'r') as file:
    content = file.read()
print(content)

多线程和多进程

Python 支持多线程和多进程编程，可以并发执行多个任务。

import threading

def print_numbers():
    for i in range(10):
        print(i)

thread = threading.Thread(target=print_numbers)
thread.start()
thread.join()

网络编程

Python 提供了多种库用于网络编程，如 socket 和 requests。

import socket

def get_ip():
    hostname = socket.gethostname()
    return socket.gethostbyname(hostname)

print(get_ip())

数据库操作

Python 可以使用 sqlite3 库操作SQLite数据库。

import sqlite3

conn = sqlite3.connect('example.db')
c = conn.cursor()
c.execute('''CREATE TABLE IF NOT EXISTS users (id INTEGER PRIMARY KEY, name TEXT)''')
c.execute('''INSERT INTO users (name) VALUES ('Alice')''')
conn.commit()
conn.close()

单元测试

单元测试用于测试代码的各个部分是否按预期工作。

import unittest

def add(a, b):
    return a + b

class TestMath(unittest.TestCase):
    def test_add(self):
        self.assertEqual(add(1, 2), 3)

if __name__ == '__main__':
    unittest.main()

数据可视化

Python 提供了多种库用于数据可视化，如 matplotlib 和 seaborn。

import matplotlib.pyplot as plt

x = [1, 2, 3, 4]
y = [10, 20, 25, 30]

plt.plot(x, y)
plt.xlabel('X axis')
plt.ylabel('Y axis')
plt.title('Simple Plot')
plt.show()

虚拟环境和包管理

虚拟环境用于隔离项目的依赖，pip 是Python的包管理工具。

# 创建虚拟环境
python -m venv myenv

# 激活虚拟环境 (Windows)
myenv\Scripts\activate

# 激活虚拟环境 (Linux/Mac)
source myenv/bin/activate

# 安装包
pip install requests

# 列出已安装包
pip freeze
