CSS 介绍

CSS（Cascading Style Sheets，层叠样式表）是一种用于描述HTML或XML文档外观和格式的样式表语言。




选择器

CSS选择器用于选取需要添加样式的HTML元素。常见的选择器包括元素选择器、类选择器和ID选择器。

/* 元素选择器 */
p {
    color: blue;
}

/* 类选择器 */
.intro {
    font-size: 20px;
}

/* ID选择器 */
#header {
    background-color: yellow;
}

盒子模型

CSS盒子模型用于描述HTML元素的布局，包括内容（content）、内边距（padding）、边框（border）和外边距（margin）。

div {
    width: 100px;
    padding: 10px;
    border: 5px solid black;
    margin: 20px;
}

Flexbox

Flexbox布局模型用于创建灵活的响应式布局结构。

.container {
    display: flex;
    justify-content: space-between;
}

.item {
    flex: 1;
}

Grid 布局

CSS Grid布局是一种强大的二维布局系统。

.grid-container {
    display: grid;
    grid-template-columns: auto auto auto;
    gap: 10px;
}

.grid-item {
    background-color: lightblue;
    padding: 20px;
    text-align: center;
}

排版

CSS排版属性用于设置字体、大小、颜色和行高等。

body {
    font-family: Arial, sans-serif;
    font-size: 16px;
    color: #333;
    line-height: 1.5;
}

背景

CSS背景属性用于设置元素的背景颜色、图像等。

body {
    background-color: #f0f0f0;
    background-image: url('background.png');
    background-repeat: no-repeat;
    background-size: cover;
}

边框

CSS边框属性用于设置元素的边框样式、宽度和颜色。

p {
    border: 1px solid black;
    border-radius: 5px;
}

动画

CSS动画用于创建逐帧动画效果。

@keyframes example {
    from {background-color: red;}
    to {background-color: yellow;}
}

div {
    width: 100px;
    height: 100px;
    background-color: red;
    animation-name: example;
    animation-duration: 4s;
}

过渡

CSS过渡用于在属性值变化时控制动画效果。

div {
    transition: width 2s;
}

div:hover {
    width: 200px;
}

媒体查询

CSS媒体查询用于根据不同设备条件应用不同的样式。

@media screen and (max-width: 600px) {
    body {
        background-color: lightblue;
    }
}

响应式设计

响应式设计通过使用灵活的网格布局、灵活的图像和CSS媒体查询来适应不同的设备和屏幕尺寸。

.container {
    max-width: 100%;
    padding: 0 20px;
}

img {
    max-width: 100%;
    height: auto;
}

伪类

CSS伪类用于选择元素的特殊状态。

a:hover {
    color: red;
}

input:focus {
    border-color: blue;
}

伪元素

CSS伪元素用于选择元素的某个部分。

p::first-line {
    color: blue;
    font-weight: bold;
}

div::before {
    content: "Note: ";
    color: red;
}

CSS 变量

CSS变量（自定义属性）用于存储属性值，以便在多个地方重复使用。

:root {
    --main-bg-color: lightblue;
}

body {
    background-color: var(--main-bg-color);
}

CSS 函数

CSS函数用于执行计算或操作，如 calc()、var() 等。

div {
    width: calc(100% - 50px);
    background-color: var(--main-bg-color);
}

优先级

CSS优先级用于确定应用哪条规则。

/* 优先级顺序：ID > 类 > 元素 */
#header {
    color: red;
}

.intro {
    color: blue;
}

p {
    color: green;
}

定位

CSS定位属性用于定位元素。

div {
    position: relative;
    top: 10px;
    left: 20px;
}

变换

CSS变换属性用于旋转、缩放、倾斜或平移元素。

div {
    transform: rotate(45deg);
}

滤镜

CSS滤镜属性用于应用图像效果，如模糊、灰度等。

img {
    filter: grayscale(100%);
}
