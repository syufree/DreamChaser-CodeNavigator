C++ 基础教程

变量

在 C++ 中，变量用于存储数据值。使用赋值运算符 = 进行赋值。

int my_variable = 10;
std::cout << my_variable << std::endl;

练习

创建一个变量 my_name 并将其设置为您的名字，然后打印该变量。


函数

函数是执行特定任务的一段代码。可以使用 return 关键字定义函数的返回值。

int add(int a, int b) {
    return a + b;
}

std::cout << add(5, 3) << std::endl;

练习

定义一个函数 multiply，接受两个参数并返回它们的积。然后调用该函数并打印结果。


条件语句

条件语句根据条件的真假执行不同的代码块。

if (x > 0) {
    std::cout << "x is positive" << std::endl;
} else if (x == 0) {
    std::cout << "x is zero" << std::endl;
} else {
    std::cout << "x is negative" << std::endl;
}

练习

编写条件语句判断一个数是奇数还是偶数。


循环

循环用于重复执行某段代码。

for (int i = 0; i < 5; ++i) {
    std::cout << i << std::endl;
}

练习

使用循环打印 1 到 10 的所有数字。
