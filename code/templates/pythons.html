Python 基础教程

Python 简介

Python 是一种解释型、高级编程语言，因其简洁、易读、功能强大而广受欢迎。它被广泛应用于 Web 开发、数据分析、人工智能、科学计算等领域。

print("Hello, World!")

变量

在 Python 中，变量用于存储数据值。使用赋值运算符 = 进行赋值。

my_variable = 10
print(my_variable)

函数

函数是执行特定任务的一段代码。可以使用 def 关键字定义函数。

def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))

条件语句

条件语句根据条件的真假执行不同的代码块。

if x > 0:
    print("x is positive")
elif x == 0:
    print("x is zero")
else:
    print("x is negative")

循环

循环用于重复执行某段代码。

for i in range(5):
    print(i)

列表

列表是一个有序的集合，可以存储多个值。

fruits = ["apple", "banana", "cherry"]
print(fruits[1])

集合

集合是一个无序的、不重复的元素集合。使用花括号 {} 定义集合。

fruits = {"apple", "banana", "cherry"}
print(fruits)

元组

元组是一个有序的、不可修改的集合。使用圆括号 () 定义元组。

fruits = ("apple", "banana", "cherry")
print(fruits[1])

异常处理

异常处理用于捕获和处理程序运行时的错误，避免程序崩溃。使用 try、except 块来捕获和处理异常。

try:
    result = 10 / 0
except ZeroDivisionError:
    print("Cannot divide by zero")

类和对象

类是面向对象编程的基础。类可以创建对象，包含数据（属性）和行为（方法）。使用 class 关键字定义类。

class Person:
    def __init__(self, name, age):
        self.name = name
        self.age = age

    def greet(self):
        return f"Hello, my name is {self.name}"

person = Person("Alice", 25)
print(person.greet())

文件操作

Python 提供了多种方式来读取和写入文件。使用 open() 函数打开文件，然后使用 read() 或 write() 方法进行操作。

with open("example.txt", "w") as file:
    file.write("Hello, World!")

with open("example.txt", "r") as file:
    content = file.read()
    print(content)

列表推导式

列表推导式是生成列表的简洁方式。使用方括号 [] 包含表达式和循环生成元素。

squares = [x**2 for x in range(10)]
print(squares)

生成器

生成器用于生成一系列值，每次请求返回一个值，使用 yield 关键字定义生成器。

def generate_numbers():
    for i in range(5):
        yield i

for number in generate_numbers():
    print(number)
